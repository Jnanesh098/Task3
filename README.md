# Java Developer Internship – Task 3

## Task: Control Flow Based Student Result System

## Description
This console-based Java application processes student results using control flow statements.
It accepts marks for multiple students, validates input, calculates total, percentage, grade,
and displays a result summary.

The program demonstrates the use of:
- if-else conditions
- switch statements
- loops (while & for)
- break and continue
- methods for reusable logic

---

## Features
- Accepts multiple student entries
- Validates marks (0–100)
- Calculates total marks and percentage
- Assigns grades based on percentage
- Displays performance status
- Clean and readable code structure

---

## Control Flow Used
- **if-else** → Grade calculation
- **switch** → Result status
- **while loop** → Multiple students
- **for loop** → Subject marks entry
- **break** → Exit program
- **continue** → Invalid marks handling

---

## Technologies Used
- Java (JDK 8+)
- Eclipse / IntelliJ IDEA
- Git & GitHub
